01. Seaborn 소개

- 기존 pandas에서 제공하는 그래프보다 다양하고 근사한 그래프를 그릴 수 있다

- Statistical Data Visualizaion 이라고 소개하고 있음

- 용어 정리

  • PDF(확률밀도함수) : 데이터셋의 분포를 나타낸다.

    • 특정 구간에 대한 확률값을 구하고 싶으면 해당 구간의 면적을 구하면 된다.

    • 그래프 아래의 모든 면적을 더하면 1이 된다.

    • 연속형 변수, 예를 들어 어떤 학생의 키가 173.5일 확률은 0이다. 왜냐하면 면적아래의 넓이를 특정값으로 구할 수 없기 때문이다.

    • 주사위, 동전을 던지는 예제에서는 특정 값에 대한 학률을 구할 수 있음!

02. KDE plot

- 세상에 일어나는 일들은 신기하게도 대부분 비슷한 분포를 띈다..

  • 근데 우리가 모은 데이터로 곡선을 그려보면 매끄러운 그래프가 아닌 약간 울긋불긋한 형태가 나온다.

  • KDE(Kernel Density Esitimation) plot을 이용하면 특정 분포에 가까운 그래프를 그려주기 때문에 부드러운 곡선을 그려준다!

#!pip install seaborn

import seaborn as sns
import matplotlib.pyplot as plt

03. LM plot

- 키와 몸무게의 연관성을 보기위한 방법

  • 어떤 학생의 키가 주어졋을 때 몸무게를 예측하는 그래프를 그려보자
sns.lmplot(data = df, x = "Height", y = "Weight")

  • 근데 거의 대부분 점들이 선이랑 떨어져 있다. 즉 저 직선이 뭔가 예측을 잘하는 것 같지 않음

  • 애초에 우리가 수집한 데이터에서 키와 몸무게간 그렇게 연관성이 있어 보이지 않음

- category plot의 약자!

  • os별 가격을 확인
sns.catplot(data = df, x = "os", y = "price", kind = "box")
sns.catplot(data = df, x = "os", y = "price", kind = "violin")

- 근데 각 범주별 데이터의 개수를 모르니까 그렇게 신뢰할 수 있는 그래프인지 모르겠음…

  • 다음과 같이 strip을 이용하자
sns.catplot(data = df, x = "os", y = "price", kind = "strip")

- 근데 또 궁금한거 \(\to\) 컴퓨터 프로세스를 만드는 기업별로 가격을 확인하고 싶음

sns.catplot(data = df, x = "os", y = "price", kind = "strip", hue = "processor_brand")

  • 오 intel을 쓰는 노트북이 더 많은 것 같다..

  • 근데 문제점 : 점들이 많이 모여있는 부분은 많이 겹쳐있어서 보기 좀 불편

sns.catplot(data = df, x = "os", y = "price", kind = "swarm", hue = "processor_brand")

- 흠…. 뭔가 되게 징그럽다… 이거를 report로 넣기엔 개인차가 많이 갈릴 것 같음
